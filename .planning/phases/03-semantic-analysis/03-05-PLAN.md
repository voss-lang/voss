- T-03-05-A: Parser/analyzer contract drift causes silent skipped checks because visitor dispatch misses actual node names.
- T-03-05-B: Example integration recomputes real embeddings during default tests.
- T-03-05-C: PRD example tension around `assistant.voss` hides strict `probable<T>` warning behavior.
- T-03-05-D: Public exports omit analyzer result types, forcing Phase 5 CLI to import private modules.

Mitigations:
- Integration tests assert exact diagnostic codes and index-file existence from parsed examples.
- Inject a fake index builder in all example tests.
- Explicitly assert `assistant.voss` emits `ANLY001` for ungated `response.value`, unless the project owner changes ANLY-01 semantics in a later planning update.
- Re-export only stable public API names from `voss/__init__.py`.
</threat_model>
