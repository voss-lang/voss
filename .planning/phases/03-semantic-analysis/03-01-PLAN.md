<objective>
Create the semantic-analysis foundation: structured diagnostics, analysis result metadata, the public `analyze()` API, a tiny AST-walker scaffold, and diagnostic tests. This is the shared contract for ANLY-01, ANLY-02, ANLY-03, Phase 4 codegen, and Phase 5 `voss check`.
</objective>

<threat_model>
Threats:
- T-03-01-A: Analyzer code accidentally executes user program expressions while walking the AST.
- T-03-01-B: Diagnostics drop source spans, making warnings impossible to audit or suppress by file/line.
- T-03-01-C: Analyzer mutates frozen AST nodes or attaches semantic state to parser-owned dataclasses.
- T-03-01-D: A high-severity internal index/path error is represented only as a warning and later codegen proceeds unsafely.

Mitigations:
- Walk dataclass AST nodes structurally only; never call user-defined functions, methods, imports, or generated Python.
- Require every `Diagnostic` to carry a Phase 2 `Span`; tests assert the file/line/column are present in `str(diagnostic)`.
- Store all analyzer state on `Analyzer` instance fields (`diagnostics`, `indexes`, scope stack); never write to AST nodes.
- Reserve `severity="error"` for unsafe analyzer conditions such as cache path escape in later plans; `AnalysisResult.ok` must be false for errors.
</threat_model>

Target state:
```python
from __future__ import annotations

from dataclasses import dataclass
from pathlib import Path
from typing import Literal

from .ast_nodes import Span

DiagnosticSeverity = Literal["warning", "error"]


@dataclass(frozen=True, slots=True)
class Diagnostic:
    severity: DiagnosticSeverity
    code: str
    message: str
    span: Span
    hint: str | None = None

    def __str__(self) -> str:
        base = (
            f"{self.span.file}:{self.span.line_start}:{self.span.col_start}: "
            f"{self.severity} {self.code}: {self.message}"
        )
        if self.hint:
            return f"{base}\n  hint: {self.hint}"
        return base


@dataclass(frozen=True, slots=True)
class EmittedIndex:
    match_id: str
    path: Path
    case_count: int
    threshold: float
    model: str


@dataclass(frozen=True, slots=True)
class AnalysisResult:
    diagnostics: tuple[Diagnostic, ...] = ()
    indexes: tuple[EmittedIndex, ...] = ()

    @property
    def warnings(self) -> tuple[Diagnostic, ...]:
        return tuple(d for d in self.diagnostics if d.severity == "warning")

    @property
    def errors(self) -> tuple[Diagnostic, ...]:
        return tuple(d for d in self.diagnostics if d.severity == "error")

    @property
    def ok(self) -> bool:
        return not self.errors
```

Do not create custom exception classes for analyzer warnings. Warnings are returned in `AnalysisResult`.
