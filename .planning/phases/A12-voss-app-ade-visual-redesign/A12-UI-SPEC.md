## Spacing Scale

8-point base. All spacing must be multiples of 4.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline label padding, pipe separators |
| sm | 8px | Pane header padding, sidebar item padding, button padding |
| md | 16px | Sidebar section padding, modal content padding |
| lg | 24px | Modal header padding, section header top margin |
| xl | 32px | Modal outer padding |
| 2xl | 48px | — (reserved, not used in A12) |
| 3xl | 64px | — (reserved, not used in A12) |

Exceptions:
- Sidebar header: 44px height (touch-target floor for `+ Agent` and collapse buttons)
- Status bar: 26px height (existing, unchanged)
- Pane header: 28px height (increased from 22px to accommodate role accent bar)
- Sidebar expand handle (collapsed state): 16px wide × 48px tall
- Collapsed sidebar width: 0px (slides fully off-screen; no icon rail)

## Typography

Three fonts. Four size tiers. Two weights per font family.

| Role | Font | Size | Weight | Line Height | Usage |
|------|------|------|--------|-------------|-------|
| Display | Poppins | 14px | 500 | 1.2 | Titlebar app/project name, sidebar section titles, modal headers |
| Display large | Poppins | 16px | 600 | 1.2 | Agent launch modal title only |
| UI body | Inter | 12px | 400 | 1.5 | Sidebar agent names, file tree entries, session history |
| UI label | Inter | 11px | 500 | 1.4 | Buttons, badges, role pills, tab labels, status bar |
| Mono body | JetBrains Mono | 11px | 400 | 1.5 | Terminal content, status bar proc name, pane header cwd/shell, model names in agent rows, cost values, timestamps, pane header dims |
| Section heading | Poppins | 11px | 600 | 1.2 | Uppercase sidebar section labels (AGENTS, FILES, SESSIONS, GIT) |

Section heading labels: uppercase, `letter-spacing: 0.08em`.

Poppins loads from Google Fonts with `wght@500;600` only — load URL: `https://fonts.googleapis.com/css2?family=Poppins:wght@500;600&display=swap`. Inter and JetBrains Mono already present.

**Collapsed tiers note:** The former 10px "Mono small" tier (model names, cost values, timestamps) is merged into the 11px "Mono body" tier. Section headings promoted from 10px to 11px. Both remain legible at dark-theme contrast ratios. Executor must verify `--focus` (#ff5b1f) on `--bg-3` (#221f1b) at 11px meets WCAG AA 4.5:1; pair with `--fg-0` label if contrast is insufficient.

## Component Contracts

### AgentSidebar

- Width: 280px, `flex-shrink: 0`. Collapses to 0 with `overflow: hidden`.
- Transition: `width 200ms cubic-bezier(0.4, 0, 0.2, 1)`. Respects `prefers-reduced-motion`.
- Left edge: 2px solid `--focus` bar via `::before` pseudo-element.
- Background: `--bg-1`.
- Border-right: `1px solid --border`.

**Sidebar header (44px):**
- Left: Voss logo mark (18px inline SVG, `--focus` fill)
- Center: "Agents" label in Poppins 500 12px `--fg-1`
- Right: `+ Agent` button (3px radius, 24×24px, orange fill on hover) + collapse chevron (`◀`) button (`aria-label="Collapse sidebar"`)

**Section structure (4 sections):** AGENTS → SESSIONS → FILES → GIT
- Section heading: Poppins 600 11px uppercase `--fg-3`, `letter-spacing: 0.08em`, padding: 8px 12px 4px
- Section body: variable height, `overflow-y: auto`

**Agent item row (32px):**
- 7px status dot with role color + `box-shadow: 0 0 4px var(--role-X)` glow
- Agent name: Inter 12px 400 `--fg-0`
- Model name: JetBrains Mono 11px `--fg-3`
- Role pill: 6px radius, 11px Poppins 600 uppercase, role background at 20% opacity
- Cost: JetBrains Mono 11px `--fg-2`, turns `--focus` when > $1.00
- Hover: `--bg-2` background
- Active (focused pane): 2px `--focus` left bar + `--focus-soft` background
- Right-click context menu items: Stop / Restart / Detach / Copy cost / Focus pane

**Streaming animation:** Pulsing orange dot — CSS keyframes `opacity 0.8s ease-in-out infinite alternate`. Stops on agent completion; dot changes to static green checkmark fading to `--fg-3` after 3 seconds.

**Collapsed state:** Sidebar width 0. A 16px × 48px expand handle appears at left edge of grid area: `--bg-2` background, `1px solid --border` right+top+bottom, `border-radius: 0 3px 3px 0`, contains `▸` glyph in `--fg-2`. Element must have `aria-label="Expand sidebar"`.

**Drag-to-reorder agents:** Visual reorder only. Implementation detail left to Claude's discretion (SolidJS-compatible approach).

### Agent Launch Modal

- Trigger: `+ Agent` button in sidebar header only (D-20). Not accessible via `⌘K`.
- Dismiss: Escape key.
- Layout: centered overlay, 480px wide, max-height 80vh, `--bg-3` background, `1px solid --border-bright` frame, `border-radius: 0` (panel style, not card).
- Backdrop: `rgba(0,0,0,0.6)`.
- Transition: 150ms fade + scale from 0.96 → 1.0. Respects `prefers-reduced-motion`.

**Modal header (48px):**
- Title: "New Agent Session" — Poppins 600 16px `--fg-0`
- Dismiss `×` button top-right, `aria-label="Dismiss"`

**CLI preset tabs (6 tabs):** Claude / Codex / Antigravity / OpenCode / Voss / Custom
- Tab style: Inter 11px 500. Active: `--focus` underline + `--fg-0`. Inactive: `--fg-2`.
- Tab row padding: 0 16px, `border-bottom: 1px solid --border`.

**Generic agent config panel** (Claude, Codex, Antigravity, OpenCode):
- Model selector: button group or dropdown (Inter 11px 500, `--bg-2` background, 3px radius)
- Effort level: Low / Medium / High segmented control
- Plan Mode toggle: checkbox/toggle (`--focus` when on)
- Skip Permissions toggle: checkbox/toggle
- Task prompt: `<textarea>` — Inter 12px 400, `--bg-2` background, `1px solid --border`, focus ring `2px solid --focus`, `border-radius: 3px`, placeholder `--fg-3`, optional (empty = interactive mode per D-18)

**Voss-specific config panel:**
- Command selector: chat / do / resume / skill / agent — segmented control
- Mode: edit / plan — toggle
- Auth choice: selector
- Task prompt: same textarea spec as above

**Custom CLI panel:**
- Name input: Inter 12px, same field style as textarea
- Command input: JetBrains Mono 12px

**Modal footer:**
- Left: "Press Ctrl+Enter to start" — Inter 11px `--fg-3`
- Right: "Launch Agent" primary button — `--focus` fill, white text, 3px radius, Poppins 500 13px. Hover: `--focus-hover`.

## Motion & Transitions

All transitions must include a `prefers-reduced-motion: reduce` override that sets duration to `0.01ms`.

| Element | Animation | Duration | Easing |
|---------|-----------|----------|--------|
| Sidebar open/close | `width` slide | 200ms | `cubic-bezier(0.4, 0, 0.2, 1)` |
| Agent launch modal appear | `opacity` 0→1 + `transform` scale 0.96→1 | 150ms | `ease-out` |
| Agent launch modal dismiss | `opacity` 1→0 + `transform` scale 1→0.96 | 100ms | `ease-in` |
| Pane focus border | `border-color`, `background` | 100ms | `ease` |
| Sidebar item hover | `background` | 80ms | `ease` |
| Agent streaming dot | `opacity` pulse | 800ms | `ease-in-out infinite alternate` |
| Done indicator fade | `opacity` 1→0 | 3s delay then 500ms | `ease` |
| Badge appear/disappear | `opacity` | 150ms | `ease` |

## Accessibility

- `prefers-reduced-motion: reduce` applied globally (already in `site/app/globals.css` pattern; replicate in `apps/voss-app/src/index.css`).
- Agent launch modal: `role="dialog"`, `aria-modal="true"`, `aria-labelledby` pointing to modal title. Focus trap on open. Focus returns to `+ Agent` button on close.
- Sidebar agent list: `role="list"` + `role="listitem"`. Active item: `aria-current="true"`.
- Streaming dot: `aria-label="Streaming"` on the pulsing element.
- Collapse chevron button: `aria-label="Collapse sidebar"`.
- Expand handle (collapsed state): `aria-label="Expand sidebar"`.
- Modal dismiss button: `aria-label="Dismiss"`.
- Context menu: `role="menu"` + `role="menuitem"`. Dismisses on Escape and on outside click.
- Focus ring: `outline: 2px solid var(--focus); outline-offset: 2px; border-radius: 2px` (inherit from site globals pattern).
- Color contrast: `--fg-0` on `--bg-0` achieves > 12:1. `--fg-2` on `--bg-0` achieves approx 4.5:1 minimum. `--focus` (#ff5b1f) on `--bg-3` (#221f1b) at 11px must be verified by executor — WCAG AA requires 4.5:1 at this size; if contrast falls short, pair the orange element with a `--fg-0` text label alongside.
