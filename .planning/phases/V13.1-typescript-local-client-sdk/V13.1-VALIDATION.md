## Sampling Rate

- **After every task commit:** Run the quick TS run (`npx vitest run`) and, for exporter tasks, the drift-gate pytest. For compile-only tasks (Waves 1–3 client code) run `npx tsc --noEmit`.
- **After every plan wave:** Run the full suite (TS + Python drift gate) + `npm run check:core-imports` + `npm run build`.
- **Before `/gsd-verify-work`:** Full suite green, including the negative drift-gate test (must FAIL on an un-snapshotted schema change) and the negative depcruise check.
- **Max feedback latency:** ~60 seconds.
