## Decisions Made

- Used source stubs for future Wave-2 modules so `src/index.ts` can be stable now and later waves can replace module contents without editing the barrel.
- Kept generated output unmodified. The generated event union is exposed as `components["schemas"]["EventEnvelope"]["event"]`, so the type test aliases that shape locally.

## Deviations from Plan

- `openapi-typescript` did not emit a named `components["schemas"]["AgentEvent"]` schema. It generated the discriminated event union inline as `components["schemas"]["EventEnvelope"]["event"]`. A1 still passed because every event member has a literal `type` and the 21-case exhaustive switch compiles.
