<threat_model>
Threats:
- T-05-05-A: The repo claims native GitHub Linguist support that does not exist yet.
- T-05-05-B: `.gitattributes` differs from the required Voss override and silently breaks TOOL-01.
- T-05-05-C: Samples drift from parser-supported Voss syntax or become trivial hello-world files unsuitable for future Linguist submission.
- T-05-05-D: Tooling tests execute samples and accidentally call providers or network services.

Mitigations:
- Metadata and README-like wording must say draft/local/future PR only.
- Tests assert the exact `.gitattributes` line.
- Tests parse all samples and compare names against expected representative programs.
- Tooling tests parse only; they do not run `voss run` or call providers.
</threat_model>
