<objective>
Complete GEN-01 construct coverage for agents, tools, prompts, classes, and spawn/gather orchestration.
</objective>

<threat_model>
Threats:
- T-04-05-A: Tool schema generation imports or executes user functions during codegen.
- T-04-05-B: Agent `spawn/gather` lowering forgets `await gather(...)` and returns coroutine objects.
- T-04-05-C: Prompt inheritance emits the wrong parent/child order, changing agent behavior.
- T-04-05-D: Pydantic model generation imports `BaseModel` for every program, violating import minimality.

Mitigations:
- `@tool` codegen only emits the decorator; runtime schema introspection happens when generated Python runs.
- Tests assert exact `await gather(handles, timeout=60)` strings.
- Prompt tests assert `CHILD_PROMPT = BASE_PROMPT + "\n" + ...`.
- Import tests in this plan assert `from pydantic import BaseModel` appears only when `ClassDecl` exists.
</threat_model>

  <action>
Implement agents, tools, prompts, classes, and gather lowering in `voss/codegen.py`.

Decorators/tools:
- Detect `FnDecl.decorators` containing `Decorator(name="tool")`.
- Add runtime import `tool`.
- Emit `@tool` immediately above `async def`.
- Do not inspect function body/docstring beyond AST fields available from parser.

AgentDecl:
- Add runtime import `VossAgent`.
- Emit:
```python
class Researcher(VossAgent):
    system_prompt = "..."
    tools = (webSearch,)
    model = "..."        # only when option present
    retries = 3          # only when option present
    async def run(self, topic: str) -> str:
        ...
```
- If body is empty, emit `pass` inside `run`.
- If return type is a generated `ClassDecl`, emit `return_type = Report` as a class attr when that information is available.

PromptDecl:
- Convert prompt names to uppercase snake constant names ending `_PROMPT`, e.g. `BaseAssistant` -> `BASE_ASSISTANT_PROMPT`.
- `PromptDecl.body` string literals concatenate with `"\n"`.
- If `extends` is present, emit `{CHILD}_PROMPT = {PARENT}_PROMPT + "\n" + {child_text!r}`.
- Preserve declaration order; if parent appears after child, raise `CodegenError("prompt parent must be declared before child")` for v1.

ClassDecl:
- Add `from pydantic import BaseModel` only when at least one class exists.
- Emit:
```python
class Report(BaseModel):
    content: str
```
- Empty classes emit `pass`.
- Use existing type mapping from Plan 04-02.

Gather:
- Detect `Call(Identifier("gather"), ...)` in await-capable contexts.
- Add runtime import `gather`.
- Emit `await gather(handles, timeout=N)` when named arg `timeout` is present.
- Normalize timeout units:
  - `60s` -> `60`
  - `500ms` -> `0.5`
  - plain numeric -> same numeric
- Do not await `spawn`; keep `Researcher().spawn(topic)`.
