### `sdk/go/sse.go` (service, SSE consumer — channel delivery)

**Analog:** `crates/voss-tui/src/net.rs` `stream_events` (lines 205-237)

**Behavioral spec from analog** (lines 205-237):
```rust
pub async fn stream_events(&self, sid: &str, tx: mpsc::Sender<AppEvent>) -> Result<()> {
    let resp = self
        .auth(self.inner.get(format!("{}/session/{}/events", self.base, sid))
            .header("Accept", "text/event-stream"))
        .send().await?;
    let resp = ok_or_detail(resp).await?;
    let mut es = resp.bytes_stream().eventsource();  // Go: bufio.Reader line parser
    while let Some(item) = es.next().await {
        match item {
            Ok(frame) => {
                // frame.event = type string (e.g. "server.connected")
                // frame.data  = JSON payload
                let ev = AppEvent::from_wire(&frame.event, &frame.data);
                let stop = matches!(ev, AppEvent::SessionIdle);
                if tx.send(ev).await.is_err() { break; }  // Go: ch <- typed; ctx.Done() -> close
                if stop { break; }
            }
            Err(e) => { tx.send(AppEvent::Error(...)).await; break; }
        }
    }
}
```

**Key differences for Go (D-03, D-04):**
- Delivery: `<-chan TypedEvent` not `mpsc::Sender` — goroutine writes to buffered channel, caller reads
- Parse: `bufio.Reader` line-by-line (hand-parsed) not `eventsource-stream` crate
- Cancel: `http.NewRequestWithContext(ctx, ...)` — context cancel tears down TCP read, goroutine exits, channel closed via `defer close(ch)`
- SSE frame lines to parse: `event: <type>`, `data: <json>`, `id: <seq>`, blank-line separator; ignore `: ping` comments

**Go target pattern** (from RESEARCH.md Pattern 3):
```go
func (c *Client) Events(ctx context.Context, sessionID string) (<-chan TypedEvent, error) {
    req, _ := http.NewRequestWithContext(ctx, "GET",
        c.baseURL+"/session/"+sessionID+"/events", nil)
    req.Header.Set("Authorization", "Bearer "+c.token)
    req.Header.Set("Accept", "text/event-stream")
    resp, err := c.http.Do(req)
    if err != nil { return nil, err }
    if resp.StatusCode != http.StatusOK { resp.Body.Close(); return nil, parseErrorBody(resp) }
    ch := make(chan TypedEvent, 32)
    go func() {
        defer resp.Body.Close()
        defer close(ch)  // guarantees no goroutine leak
        parseSSEStream(ctx, resp.Body, ch)
    }()
    return ch, nil
}
```

**Behavioral spec** (lines 24-37):
```rust
async fn ok_or_detail(resp: reqwest::Response) -> Result<reqwest::Response> {
    let status = resp.status();
    if status.is_success() { return Ok(resp); }
    let body = resp.text().await.unwrap_or_default();
    let detail = serde_json::from_str::<Value>(&body)
        .ok()
        .and_then(|v| v.get("detail").and_then(|d| d.as_str()).map(String::from));
    match detail {
        Some(d) => Err(anyhow!("{d} ({status})")),
        None    => Err(anyhow!("HTTP {status}")),
    }
}
```

**Go translation** (from RESEARCH §Code Examples):

```go
type VossError struct { Status int; Detail string }
func (e *VossError) Error() string {
    return fmt.Sprintf("HTTP %d: %s", e.Status, e.Detail)
}

func checkResponse(resp *http.Response) error {
    if resp.StatusCode/100 == 2 { return nil }
    defer resp.Body.Close()
    body, _ := io.ReadAll(resp.Body)
    var payload struct { Detail string `json:"detail"` }
    _ = json.Unmarshal(body, &payload)
```

```go
type Handshake struct {
    V     int    `json:"v"`
    Port  uint16 `json:"port"`
    Token string `json:"token"`
}

func readHandshake(r io.Reader, timeout time.Duration) (Handshake, error) {
    ch, errCh := make(chan Handshake, 1), make(chan error, 1)
    go func() {
        scanner := bufio.NewScanner(r)
        for scanner.Scan() {
            var h Handshake
            if err := json.Unmarshal(scanner.Bytes(), &h); err == nil && h.Token != "" {
                ch <- h; return
            }
        }
        // scanner.Err() == nil means EOF without handshake
        if err := scanner.Err(); err != nil { errCh <- err } else {
            errCh <- errors.New("server exited before handshake")
        }
    }()
    select {
    case h := <-ch: return h, nil
    case err := <-errCh: return Handshake{}, err
    case <-time.After(timeout): return Handshake{}, errors.New("handshake timeout")
    }
}
```

## Shared Patterns

### Bearer Auth Injection
**Source:** `crates/voss-tui/src/net.rs` lines 57-59 (`auth()` method)
**Apply to:** Every request in `rest.go` AND the SSE GET in `sse.go`
```rust
fn auth(&self, rb: RequestBuilder) -> RequestBuilder {
    rb.bearer_auth(&self.token)  // Go: req.Header.Set("Authorization", "Bearer "+c.token)
}
```
Every outgoing `http.Request` — whether REST or SSE GET — must carry this header before `c.http.Do(req)`.

### Error Body Decode
**Source:** `crates/voss-tui/src/net.rs` lines 24-37 (`ok_or_detail`)
**Apply to:** Every REST method in `rest.go` + SSE status check in `sse.go`
Non-2xx → unmarshal `{v, detail}` body → `&VossError{Status: resp.StatusCode, Detail: payload.Detail}`.

### Python Interpreter Resolution
**Source:** `crates/voss-tui/src/server.rs` lines 37-47 (`python_path()`) + `crates/voss-providers/tests/schema_parity.rs` lines 24-33 (`pick_python()`)
**Apply to:** `client.go` `interpreterPath()` + `drift_test.go` Python invocation
Chain: `VOSS_PYTHON` env var → `../../.venv/bin/python` relative path → `"python3"` fallback. Always check existence before returning the `.venv` path.
