## Deviation (1, justified)

**Handshake timeout 20s → 60s + `LITELLM_LOCAL_MODEL_COST_MAP=true` injected.** Plan specified a 20s handshake. `voss serve` eagerly imports litellm; its cold import (remote cost-map fetch ~12s + import-tree compile) can exceed 20s — the exact failure V13.2-06 diagnosed and fixed the same way. `Spawn` sets `LITELLM_LOCAL_MODEL_COST_MAP=true` (drops the network fetch) and uses a 60s budget. Bad-interpreter still fails instantly (exec error at `Start`, not a timeout), so the no-hang guarantee holds regardless.
