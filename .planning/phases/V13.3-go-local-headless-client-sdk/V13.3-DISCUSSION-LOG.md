## SSE-delivery idiom + dependency policy

| Option | Description | Selected |
|--------|-------------|----------|
| `<-chan Event` + context cancel, stdlib only | Channel delivery; ctx cancel = disconnect-cancels §8; stdlib net/http, SSE hand-parsed | ✓ |
| `iter.Seq2[Event,error]` range-over-func | Go 1.23 iterator; requires 1.23+ floor | |
| Callback handler `func(Event) error` | SDK drives the loop | |

**User's choice:** `<-chan Event` + context cancel, stdlib only (recommended)
**Notes:** Most composable for CI-bot/headless audience; avoids a hard Go 1.23 floor. stdlib-only matches the "client wiring hand-written, codegen = types only" carry-forward.
