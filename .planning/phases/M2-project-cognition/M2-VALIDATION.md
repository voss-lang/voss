| **Full suite command** | `pytest -q --strict-markers` |
| **Estimated runtime** | ~30s (harness subset); ~60s (full) |

## Sampling Rate

- **After every task commit:** Run `pytest tests/harness/ -x`
- **After every plan wave:** Run `pytest -q --strict-markers`
- **Before `/gsd-verify-work`:** Full suite must be green
- **Max feedback latency:** ~30 seconds (quick); ~60 seconds (full)
