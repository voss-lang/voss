## What shipped

- `docs/native-embedding.md` (133 lines) — single prose reference for native/C embedders, five sections:
  - **(1) Consumption model** — spawn/attach `voss serve`; read the one-line `{v,port,token}` stdout handshake; connect over loopback `http://127.0.0.1:<port>` (loopback-only, no TLS); `Authorization: Bearer <token>` on every request incl. the SSE GET; REST commands + SSE (`text/event-stream`) events; one turn/session → 409. Plus a token-handling security subsection (ephemeral/per-process, never log, never bind routable).
  - **(2) Contract source-of-truth** — cites `.planning/PROTOCOL.md`, `contracts/openapi.json`, `contracts/events.schema.json` by path; cross-links `docs/sdk.md` + `docs/ORCHESTRATION_LAYERS.md` (and the current design source `.planning/docs/ORCHESTRATION_LAYERS.md`); states it points-at, never forks, the schema.
  - **(3) JSON→native reading table** — `type` discriminator (= SSE event name = serde tag), `v` version field, scalar/composite reading, required-vs-optional, discriminated-union decode; member set declared **authoritative-from `contracts/events.schema.json`** (not a hardcoded list).
  - **(4) Stability statement** — contract stable-as-PROTOCOL.md-v1 + `v`-bump policy; maps the protocol surface to **generated-from-protocol**, the wire contract to **stable-now**, C headers/FFI to **deferred**; uses the five tier names verbatim (stable-now / experimental / generated-from-protocol / private-internal / deferred).
  - **(5) Deferred** — C headers, cbindgen, FFI/cgo, high-level C ergonomics marked deferred as a complete end-state; activation trigger recorded (a concrete native embedder with a real need appears).
- `docs/check-native-embedding-refs.sh` (executable, POSIX, no deps) — enumerates all five cited paths; hard-fails on a missing always-expected path (`PROTOCOL.md`/`sdk.md`); warn-skips (exit 0) while any upstream-gated path (`contracts/*.json` V13.1, `docs/ORCHESTRATION_LAYERS.md` V13) is absent; greps `generated-from-protocol` + `deferred` in `docs/ORCHESTRATION_LAYERS.md` once all upstream-gated paths resolve.
